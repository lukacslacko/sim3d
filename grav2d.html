2D gravity v2 — sourceless flow + prescribed curl

M is a 2D mass-flow field on a periodic N×N torus, kept sourceless (∇·M = 0); ω is a scalar curl field and we want curl(M) = ω. At each step we (1) recompute the target curl via the literal kernel ωtarget(q) = κ · Σp [m(p) · (q−p)] / |q−p|² (or the cross-product variant) using minimum-image distances on the torus; (2) relax ω toward ωtarget; (3) push curl(M) toward ω by adding ∇⊥ψ where −∇²ψ = ω − curl(M); (4) project M onto the divergence-free subspace via M ← M − ∇χ, ∇²χ = ∇·M; (5) rescale M so Σ|M| equals its value at Randomize.
Caveat re. the dot kernel: on the torus its convolution is morally ∇−2(∇·M), so once M is div-free it returns ωtarget=0 and the system relaxes to uniform flow. The cross variant returns the smoothed self-curl ∇−2(curl M) which is non-trivial — try it. κ is signed so you can flip the orientation of either kernel.

     
Kernel sum is O(N⁴) per step — at N=24 it's a few hundred kops/frame, smooth. Cross-product variant integrates m_x(q_y−p_y) − m_y(q_x−p_x). ω relax rate · dt ∈ [0,1] is the fraction by which ω moves toward target each step; curl-fix rate · dt ∈ [0,1] is the fraction of the curl mismatch we remove each step. Self-pair (q=p) is excluded from the kernel.
